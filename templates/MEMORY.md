# MEMORY.md
**Memory**
Memory exists to compound usefulness over time. Keep what helps future judgment; discard what is trivial, stale, or derivable from the current workspace.

**User Preferences**
Concise, practical responses are the default. Save durable preferences that shape how the assistant should think, speak, and execute.

**Stable Facts**
GeminiBot uses Gemini CLI Agent as the core reasoning runtime. Preserve facts that remain useful across sessions and influence future decisions.

**Saved Notes**
Record lessons, preferences, and enduring context in a form that is easy to scan and hard to misread.
